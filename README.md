# B站字幕提取器

一个简单易用的 Windows 图形界面工具，用于从B站视频自动提取字幕并生成格式化的 Markdown 文稿。

## 功能特性

- 🎬 支持粘贴B站视频网址自动识别
- 📝 自动提取视频字幕内容
- 🏷️ 按字幕自然段落自动分章节
- 🎯 支持多种字幕语言选择
- 📄 输出格式自定义（Markdown、纯文本等）
- 💾 自定义输出文件夹位置
- ✨ 清理/格式化输出，无时间戳

## 使用方法

### 最简单的方式（推荐）
1. 下载 `bilibili-subtitle-extractor.exe`
2. 双击运行
3. 粘贴B站视频网址（如：`https://www.bilibili.com/video/BV...`）
4. 选择字幕语言和输出格式
5. 选择保存位置
6. 点击"开始提取"按钮

### 从源代码运行
```bash
python -m pip install -r requirements.txt
python main.py
```

## 系统要求

- Windows 7 或更新版本
- Python 3.8+ (如果从源代码运行)
- 联网环境
- 足够的硬盘空间用于临时文件

## 常见问题

**Q: 为什么有些视频没有字幕？**
A: 部分B站视频没有上传字幕文件，工具无法提取。请确保视频有可用的字幕。

**Q: 支持哪些语言的字幕？**
A: 支持视频本身提供的所有字幕语言（通常包括中文、英文等）。

**Q: 输出的文件在哪里？**
A: 在你选择的输出文件夹中，文件名为 `[视频标题].md`

**Q: 工具卡住了怎么办？**
A: 请稍候，大文件提取可能需要几分钟。如果长时间无反应，请检查网络连接。
